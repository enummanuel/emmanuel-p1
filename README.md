# Project 1: My First Personal Web Page
A simple personal website built as the Week 1 Session 1 homework assignment for Outcome School.
## Project Overview
This project demonstrates the core relationship between writing raw code and how a web browser processes it to render a live page.
## Tech Stack Used
* **HTML**: Structured the headings, text introduction, and image elements.
* **CSS**: Added background colors, custom text styling, and basic formatting.
* **JavaScript**: Verified console initialization using Chrome DevTools.
## What I Learned
* How to correctly set up a local development directory structure in VS Code.
* Linking external `.css` style sheets and `.js` scripts to an `index.html` skeleton.
* Inspecting live web pages using the browser console to verify script execution.
